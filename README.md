# PawMart

**Live Site:** [https://your-live-site-url.example.com](https://your-live-site-url.example.com)

PawMart connects local pet owners and buyers for pet adoptions and care products. Built with **React**, **Firebase**, and **MongoDB**, it promotes adopting pets instead of buying.

##  Features

*  Adopt-first marketplace for pets & supplies
*  Firebase Auth (Email/Password + Google Login)
* Ш Add, Update & Delete Listings (Private Routes)
* Ь View My Orders & Export PDF Report (jsPDF)
*  Fully Responsive & Toast Notifications

##  Pages

* **Home:** Banner, Categories, Recent Listings, Awareness & Heroes Sections
* **Pets & Supplies:** All listings with category filter
* **Add Listing:** Form for pet or product uploads (Private)
* **Listing Details:** Show details + Order Modal (Private)
* **My Listings / My Orders:** User dashboards
* **Login / Register:** Firebase authentication
* **404 Page:** Shown without Navbar & Footer

## 锔 Tech Stack

React 路 Tailwind CSS 路 Firebase 路 Node.js 路 Express 路 MongoDB 路 jsPDF

##  Setup

```bash
# Client
cd client && npm install && npm run dev
# Server
cd server && npm install && npm run dev
```

Add your `.env` files with Firebase & MongoDB credentials.
